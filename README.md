# Bookue-front

Please, use mobile screen on browser to see correct stylization.

## Requirements
- Node.js
- Docker (with docker compose)

## How to install
```
yarn
// or
npm install
```

## How to run
```
docker-compose up -d
```

## How to tests
If you want run unit tests, run:
```
yarn test:unit
```
If you want run e2e tests, run:
```
yarn e2e:run
```
